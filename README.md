
# Module 9A - Build Your First React Component

## Description
This is a React app created using Vite that renders a UserProfile component. The component includes a name, bio, and image styled with inline CSS.
EXTRA: I changed UserProfile to take props so that I could render multiple instances of the UserProfile component with different people.

## What I Learned
- How to set up a React project using Vite
- How to write and structure a functional React component
- How to use JSX and inline styles
- How to render a component in App.jsx

## Challenges
Briefly describe any errors, bugs, or confusion you encountered and how you solved it.

I started learning about React a little early and did some solo-work following a separate tutorial before working on this assignment, so I didn't face much confusion for this project specifically. When I was first installing Node.js and making my first Vite project, I encountered some confusion regarding the difference between Oxlint and ESlint and wasn't sure how to open the created project in the browser, but I just experimented until things started falling into place. Excited to do more with this!
